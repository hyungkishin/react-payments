  - [ ] 유효성 검증 실패에 대한 UI/UX 추가

- [ ] 카드

  - [ ] 카드사를 선택하지 않아도 모달을 닫을 수 있다.
  - [ ] 카드사가 선택되고 유효한 카드 번호 16자리를 모두 입력하면, 자동으로 만료일로 focus된다.
